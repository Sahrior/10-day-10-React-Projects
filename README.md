# 10 Days 10 React Projects 🚀

A collection of 10 beginner-friendly React projects built in 10 days. This repository is mainly created for **practice** and to help **junior developers** learn React by building real projects step by step.

## Features ✨

* 📦 10 different mini React projects  
* ⚛️ Hands-on practice with React fundamentals  
* 🔁 Reusable components and patterns  
* 🎯 Beginner-friendly code structure  
* 📱 Responsive UI in most projects  
* ⚡ Fast development using Vite  

---

## Purpose 🎯

This project is built to:

* Practice React by building real projects daily  
* Strengthen fundamentals through repetition  
* Help juniors learn by exploring simple projects

## Installation ⚙️

Clone the repository:

```bash
git clone https://github.com/your-username/10-day-10-react-projects.git
```

Go to the project directory:

```bash
cd 10-day-10-react-projects
```

Install dependencies (for each project):

```bash
npm install
```

Run any project:

```bash
npm run dev
```

---

## Available Scripts 📜

```bash
npm run dev
```

Runs the app in development mode.

```bash
npm run build
```

Builds the app for production.

```bash
npm run preview
```

Preview the production build.
